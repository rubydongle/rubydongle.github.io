UART 0 的寄存器

1.UART线路控制寄存器(UART Line Control Register)
寄存器的地址在0x50000000

Register Address R/W Description Reset Value
ULCON0 0x50000000 R/W UART channel 0 line control register 0x00

每个位的作用如下：

ULCONn Bit Description Initial State
Reserved [7] - 0
Infrared Mode [6] 是否使用Infrared mode.
0 = Normal mode operation
1 = Infrared Tx/Rx mode		 0
Parity Mode [5:3] Specify the type of parity generation and checking during UART transmit and receive operation.
0xx = No parity
100 = Odd parity
101 = Even parity
110 = Parity forced/checked as 1
111 = Parity forced/checked as 0		 000
Number of Stop Bit [2] Specify how many stop bits are to be used for end-of-frame signal.
0 = One stop bit per frame
1 = Two stop bit per frame		 0
Word Length [1:0] Indicate the number of data bits to be transmitted or received per frame.
00 = 5-bits 01 = 6-bits
10 = 7-bits 11 = 8-bits		 00

2.UART控制寄存器(UART Control Register)
寄存器的地址在0x50000004

Register Address R/W Description Reset Value
UCON0 0x50000004 R/W UART channel 0 control register 0x00

3.UART FIFO控制寄存器(UART FIFO Control Register)
寄存器地址在0x50000008

Register Address R/W Description Reset Value
UFCON0 0x50000008 R/W UART channel 0 FIFO control register 0x00

4.UART调制解调器控制寄存器(UART Modem Control Register)
寄存器地址在0x5000000C

Register Address R/W Description Reset Value
UMCON0 0x5000000C R/W UART channel 0 Modem control register 0x00

5.UART 发送/接收状态寄存器(UART TX/RX Status Register)
寄存器地址在0x50000010

Register Address R/W Description Reset Value
UTRSTAT0 0x50000010 R UART channel 0 Tx/Rx status register 0x00

6.UART 错误状态寄存器(UART Error Status Register)
寄存器的地址在0x50000014

Register Address R/W Description Reset Value
UERSTAT0 0x50000014 R UART channel 0 Rx error status register 0x0

7.UART FIFO 状态寄存器(UART FIFO Status Register)
寄存器的地址在0x50000018

Register Address R/W Description Reset Value
UFSTAT0 0x50000018 R UART channel 0 Rx FIFO status register 0x0

8.UART调制解调器状态寄存器(UART Modem Status Register)
寄存器地址在0x5000001C

Register Address R/W Description Reset Value
UFSTAT0 0x5000001C R UART channel 0 Rx modem status register 0x0

9.UART 发送缓冲寄存器(UART TRANSMIT BUFFER REGISTER (HOLDING REGISTER & FIFO REGISTER))

Register Address R/W Description Reset Value
UTXH0 0x50000020(L)
0x50000023(B)		 W
(by byte)		 UART channel 0 transmit buffer register -

10.UART接收缓冲区寄存器(UART RECEIVE BUFFER REGISTER (HOLDING REGISTER & FIFO REGISTER))

Register Address R/W Description Reset Value
URXH0 0x50000024(L)
0x50000027(B)		 R
(by byte)		 UART channel 0 receive buffer register -

11.UART波特率分频因子寄存器(UART BAUD RATE DIVISOR REGISTER)

Register Address R/W Description Reset Value
UBRDIV0 0x50000028 R/W Baud rate divisior register 0 -
